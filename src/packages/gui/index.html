Wie kann ich ein neues Projekt anlegen

  1. Öffne das Projekt über den linken Bereich
  2. Klicke auf NEU neben "Literatureinträge"
  3. Vergebe eine ID und wähle den gewünschten Literaturtypen aus
  4. Fülle die Felder aus
  5. Klicke speichern
  1. Öffne das Projekt über den linken Bereich
  2. Klicke auf "Zitate aufräumen"
  3. Nun musst du diesen Button immer klicken, wenn du eine neue Quelle zum ersten Mal zitierst!

Wenn wieder alle Quellen gelistet werden sollten, schreibe in die "citedKeys.csv" im Projektordner ein *

Die Literaturtypen werden pro Projekt gepflegt. Um einen neuen zu erstellen befolge diese Schritte:

  1. Öffne das Projekt über den linken Bereich
  2. Klicke auf NEU neben "Literaturtypen"
  3. Vergebe einen Namen und gib die gewünschten Felder für das Literaturverzeichnis an
  4. Gib die gewünschten Felder für Zitate an
  5. Vergiss nicht, eventuell gewüschte Präfixe oder Suffixe anzugeben
  6. Die Attribute können auch kursiv dargestellt werden
  7. Klicke speichern

Kopiere das Element aus der "literature_types.json" in die JSON-Datei in deinem Projekt.

  1. Öffne das Projekt über den linken Bereich
  2. Exportiere die Quelle mit Citavi nach BibTex
  3. Drap-and-Droppe die Datei in den Projektbereich
  4. Überprüfe, ob die Quelle korrekt angelegt wurde (bei IEEE-Exporten kann es passieren, dass am letzten Attribut noch ein } hängt). Berichtige Fehler, gib einen Autor_kurz für Zitate an und speichere.
